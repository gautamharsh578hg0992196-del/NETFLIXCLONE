🎬 Netflix Clone – Responsive Streaming UI

A fully responsive Netflix India Clone Website built using HTML, CSS, and JavaScript.
This project recreates the modern Netflix landing page UI with smooth scrolling, trending movie cards, FAQ accordion, sticky navbar, multilingual support (English + Hindi), and responsive design.

🚀 Features
🎥 Netflix-style Hero Section
📱 Fully Responsive Design
🌐 Multi-language Support (English / Hindi)
🎬 Trending Movies Horizontal Slider
📺 Feature Sections with Videos
❓ Interactive FAQ Accordion
📩 Email Validation Form
🔝 Sticky Navbar on Scroll
✨ Smooth UI Animations
📱 Mobile Friendly Layout
🎨 Clean Netflix Inspired Design

HTML5
CSS3
JavaScript (Vanilla JS)
